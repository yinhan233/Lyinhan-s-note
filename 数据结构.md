## 绪论
1、在数据结构中，从逻辑上可以把数据结构分成<font color="#ffff00"><u>线性结构</u></font>与<font color="#ffff00"><u>非线性结构</u></font>  
2、计算机算法指的是解决问题的有限运算序列，它必须具备<u><font color="#ffff00">输入</font></u>、<font color="#ffff00"><u>输出</u></font>和  
<font color="#ffff00"><u>可行性</u></font>、<u><font color="#ffff00">确定性</font></u>和<u><font color="#ffff00">有穷性</font></u>等5个特性。 
```Markdown
1. 明确性（Definiteness）：:算法的每一步都必须有明确的定义，不能有歧义。
2. 有限性（Finiteness）：:算法必须在有限的步骤内终止，不能是无限循环。
3. 可行性（Effectiveness）：:算法的每一步都必须能够通过执行有限次数完成，即每一步操作都是可实现的。
4. 输入（Input）：:算法可以接受零个或多个输入。
5. 输出（Output）：:算法需要产生至少一个输出。
```
3、数据结构是一门研究非数值计算的程序设计问题中计算机的<u><font color="#ffff00">数据元素</font></u>以及它们之间的<u><font color="#ffff00">关系</font></u>和<u><font color="#ffff00"><font color="#ffff00">运算</font></font></u>等的学科。  
## 线性表
1、线性表是一个<font color="#ffff00">有限</font>序列，可以<font color="#ffff00">为空</font>。  
2、‍线性表采用<font color="#ffff00">链式存储</font>时，其地址<font color="#ffff00">连续与否均可以</font>。  
3、用链表表示线性表的优点是<font color="#ffff00">便于插入和删除</font>  
4、循环链表的主要优点是<font color="#ffff00">从表中的任意结点出发都能扫描到整个链表</font>  。
5、给定有n个结点的向量，建立一个<font color="#ffff00">有序单链表</font>的时间复杂度为<font color="#ffff00">O(n^2)</font>  。
6、顺序存储结构是一种<font color="#ffff00">随机存取</font>的存储结构。  
7、顺序存储的线性表可以按序号<font color="#ffff00">随机存取</font>。
```Wikipedia
随机访问（英语：random access），亦称直接访问（direct access），代表同一时间访问一组序列中的一个随意组件。
```
8、将两个各有n个元素的有序表归并成一个有序表，在最坏的情况下，其比较次数是<font color="#ffff00">2n-1</font>。
9、线性表中的元素可以是各种各样的，但同一线性表中的数据元素具有相同的特性，因此是属于<font color="#ffff00">同一数据对象</font>。
10、线性结构是一种<font color="#ffff00">一对一</font>的结构。
```Ai explain
一对一:元素间的逻辑邻接关系是'每个元素（除首尾外）仅有一个直接前驱和一个直接后继'
```
