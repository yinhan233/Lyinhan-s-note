### 角度与弧度
任意角：角可以看作是平面内一条射线绕着端点从初始位置旋转到终止位置的图形  
- 逆时针为正角，顺时针为负角  
- $1° = \frac{π}{180}\text{ 弧度}$  
- $1\text{ 弧度} = \frac{180°}{π}$  
坐标上的角：

| 象限          | 角度范围                     |
|---------------|-----------------------------|
| 第一象限       | $\{α \mid 2kπ < α < 2kπ + \frac{π}{2}\}$ |
| 第二象限       | $\{α \mid 2kπ + \frac{π}{2} < α < 2kπ + π\}$ |
| 第三象限       | $\{α \mid 2kπ + π < α < 2kπ + \frac{3π}{2}\}$ |
| 第四象限       | $\{α \mid 2kπ + \frac{3π}{2} < α < 2kπ + 2π\}$ |
| 终边在x轴      | $\{α \mid α = kπ,\ k∈\mathbb{Z}\}$ |
| 终边在坐标轴   | $\{α \mid α = \frac{kπ}{2},\ k∈\mathbb{Z}\}$ |
### 任意角的三角函数
定义：坐标系内，角$α$的顶点为原点，始边为x轴正半轴，终边上取点$P(x,y)≠O$，$r = \sqrt{x^2 + y^2}$  
- $\sinα = \frac{y}{r}$，$\cosα = \frac{x}{r}$  
- $\tanα = \frac{y}{x}$，$\cotα = \frac{x}{y}$  
- $\secα = \frac{r}{x}$，$\cscα = \frac{r}{y}$  
# 三角公式
### 同角关系
| 关系类型   | 公式                          | 约束条件                 |
|------------|------------------------------|--------------------------|
| 平方关系   | $\sin^2α + \cos^2α = 1$      |                          |
| 商数关系   | $\tanα = \frac{\sinα}{\cosα}$ | $α ≠ kπ + \frac{π}{2}$   |
|            | $\cotα = \frac{\cosα}{\sinα}$ | $α ≠ kπ$                 |
| 倒数关系   | $\tanα \cdot \cotα = 1$      | $α ≠ \frac{kπ}{2}$       |
|            | $\sinα \cdot \cscα = 1$      |                          |

### 两角和差
$$\begin{aligned}
\cos(α \pm β) &= \cosα\cosβ \mp \sinα\sinβ \\
\sin(α \pm β) &= \sinα\cosβ \pm \cosα\sinβ \\
\tan(α \pm β) &= \frac{\tanα \pm \tanβ}{1 \mp \tanα\tanβ}
\end{aligned}$$
### 倍角公式
$$\begin{aligned}
\sin2α &= 2\sinα\cosα \\
\cos2α &= \cos^2α - \sin^2α = 2\cos^2α - 1 = 1 - 2\sin^2α \\
\tan2α &= \frac{2\tanα}{1 - \tan^2α}
\end{aligned}$$
### 半角公式
$$\begin{aligned}
\sin\frac{α}{2} &= \pm\sqrt{\frac{1 - \cosα}{2}} \\
\cos\frac{α}{2} &= \pm\sqrt{\frac{1 + \cosα}{2}} \\
\tan\frac{α}{2} &= \pm\sqrt{\frac{1 - \cosα}{1 + \cosα}} = \frac{\sinα}{1 + \cosα} = \frac{1 - \cosα}{\sinα}
\end{aligned}$$
### 降幂公式
$$\begin{aligned}
\sin^2α &= \frac{1 - \cos2α}{2} \\
\cos^2α &= \frac{1 + \cos2α}{2}
\end{aligned}$$

### 辅助角公式
$α\sinα + b\cosα = \sqrt{a^2 + b^2}\sin(α + \varphi)$  
其中 $\cos\varphi = \frac{a}{\sqrt{a^2 + b^2}},\ \sin\varphi = \frac{b}{\sqrt{a^2 + b^2}},\ \tan\varphi = \frac{b}{a},\ (0<\varphi<2π)$
### 积化和差
$$\begin{aligned}
\sinα\cosβ &= \frac{1}{2}[\sin(α+β) + \sin(α-β)] \\
\cosα\sinβ &= \frac{1}{2}[\sin(α+β) - \sin(α-β)] \\
\cosα\cosβ &= \frac{1}{2}[\cos(α+β) + \cos(α-β)] \\
\sinα\sinβ &= -\frac{1}{2}[\cos(α+β) - \cos(α-β)]
\end{aligned}$$

### 和差化积
$$\begin{aligned}
\sinα + \sinβ &= 2\sin\left(\frac{α+β}{2}\right)\cos\left(\frac{α-β}{2}\right) \\
\sinα - \sinβ &= 2\cos\left(\frac{α+β}{2}\right)\sin\left(\frac{α-β}{2}\right) \\
\cosα + \cosβ &= 2\cos\left(\frac{α+β}{2}\right)\cos\left(\frac{α-β}{2}\right) \\
\cosα - \cosβ &= -2\sin\left(\frac{α+β}{2}\right)\sin\left(\frac{α-β}{2}\right)
\end{aligned}$$

### 万能公式
令 $t = \tan\frac{α}{2}$，则：
$$\begin{aligned}
\sinα &= \frac{2t}{1 + t^2}, & \cosα &= \frac{1 - t^2}{1 + t^2}, & \tanα &= \frac{2t}{1 - t^2}
\end{aligned}$$

# 解三角形
### 正弦定理
$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$  
（$R$ 为外接圆半径）

### 余弦定理
$$\begin{aligned}
a^2 &= b^2 + c^2 - 2bc\cos A \\
\cos A &= \frac{b^2 + c^2 - a^2}{2bc}
\end{aligned}$$

### 面积公式
$$\begin{aligned}
S &= \frac{1}{2}ab\sin C = \frac{abc}{4R} = rp \\
\text{其中} \quad p &= \frac{a+b+c}{2}
\end{aligned}$$
### 海伦公式
$S = \sqrt{p(p-a)(p-b)(p-c)}$  
其中 $p = \dfrac{a+b+c}{2}$

**注意**：已知两边及其中一边的对角时（SSA），解可能不唯一
