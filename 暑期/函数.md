# 函数及其表示
### 函数的相关概念
1. **定义域与值域**
    - 在函数 $y=f(x), x \in A$ 中：
    - $x$ 称为**自变量**，$x$ 的取值范围 $A$ 称为**定义域**
    - $y$ 称为**函数值**，${f(x) \mid x \in A}$ 称为**值域**
    - 定义域和值域均不为空集
2. **函数的三要素**  
    定义域、值域和对应关系
3. **相等函数**  
    定义域和对应关系完全一致的函数
### 函数的表示法
- 解析法（公式表示）
- 图象法（图形表示）
- 列表法（表格表示）
### 分段函数
- 在定义域的不同子集上，用不同的表达式表示
- 定义域：各段定义域的并集
- 值域：各段值域的并集

# 函数的性质
### 奇偶性（定义域关于原点对称）

|类型|定义式|图象特征|特殊点|
|---|---|---|---|
|奇函数|$f(-x) = -f(x)$|关于**原点对称**|$f(0)=0$ 或不存在|
|偶函数|$f(-x) = f(x)$|关于**y轴对称**|-|
|非奇非偶|不满足以上条件|-|-|

### 对称性
#### 轴对称
- $f(a-x) = f(a+x)$ ⇔ 关于 $x=a$ 对称
- $f(2a-x) = f(x)$ ⇔ 关于 $x=a$ 对称
- $f(a+x) = f(b-x)$ ⇔ 关于 $x=\frac{a+b}{2}$ 对称
#### 点对称
- $f(x) + f(-x+b) = 0$ ⇔ 关于 $(\frac{b}{2},0)$ 对称
- $f(a-cx) + f(b+cx) = 0$ ⇔ 关于 $\left(\frac{a+b}{2},0\right)$ 对称
- $f(a-cx) + f(b+cx) = c$ ⇔ 关于 $\left(\frac{a+b}{2},\frac{c}{2}\right)$ 对称
### 周期性
- **定义**：$f(x) = f(x+T) \ (T \neq 0)$，则 $T$ 为函数周期
- **最小正周期**：最小的 $T>0$
- **常用结论**：
    - $f(x+a) = -f(x)$ ⇒ $T=2a$
    - $f(x+a) = \frac{1}{f(x)}$ ⇒ $T=2a$
    - $f(x+a) = -\frac{1}{f(x)}$ ⇒ $T=2a$
    - $f(x+a) = \frac{1+f(x)}{1-f(x)}$ ⇒ $T=4a$
### 单调性

|类型|定义|等价形式|
|---|---|---|
|严格增函数|$\forall x_1 > x_2 \in D,\ f(x_1) > f(x_2)$|$\frac{f(x_1)-f(x_2)}{x_1-x_2} > 0$|
|增函数|$\forall x_1 > x_2 \in D,\ f(x_1) \geq f(x_2)$|-|
|严格减函数|$\forall x_1 < x_2 \in D,\ f(x_1) > f(x_2)$|$\frac{f(x_1)-f(x_2)}{x_1-x_2} < 0$|
|减函数|$\forall x_1 < x_2 \in D,\ f(x_1) \geq f(x_2)$|-|

#### 单调性运算性质

| 运算                       | 单调性变化规律      |
| ------------------------ | ------------ |
| $f(x) + g(x)$            | 同增或同减时保持原单调性 |
| $k \cdot f(x)\ (k>0)$    | 与原函数单调性相同    |
| $k \cdot f(x)\ (k<0)$    | 与原函数单调性相反    |
| $\frac{1}{f(x)},\ -f(x)$ | 与原函数单调性相反    |
| 复合函数 $f(g(x))$           | **同增异减**原则   |

# 函数图象变换
### 基本变换

|变换类型|表达式|图象变化|
|---|---|---|
|平移|$f(x) + k$|**上**加**下**减|
||$f(x + h)$|**左**加**右**减|
|伸缩|$f(wx)\ (0<w<1)$|横坐标伸长 $\frac{1}{w}$ 倍|
||$f(wx)\ (w>1)$|横坐标缩短 $\frac{1}{w}$ 倍|
||$kf(x)\ (k>1)$|纵坐标伸长 $k$ 倍|
||$kf(x)\ (0<k<1)$|纵坐标缩短 $k$ 倍|

### 对称变换

|表达式|对称关系|

|$-f(x)$|关于 **x轴** 对称|
|$f(-x)$|关于 **y轴** 对称|

# 幂函数
### 概念与特征
- 一般形式：$y = x^\alpha$（$\alpha$ 为常数）
- 定义域：取决于指数 $\alpha$（$x>0$ 总有定义）
- 图象必过点 $(1,1)$
### 常见幂函数图象与性质

```functionplot

yLabel: y

grid: false
---
f(x) = x
f(x) = x^2
f(x) = x^3
f(x) = sqrt(x)
f(x) = 1/x
```

|函数|定义域|值域|奇偶性|单调区间|
|---|---|---|---|---|
|$y = x$|$\mathbb{R}$|$\mathbb{R}$|**奇**|$\mathbb{R}$ 上**增**|
|$y = x^2$|$\mathbb{R}$|$[0, +\infty)$|**偶**|$(-\infty,0]$**减**，$[0,+\infty)$**增**|
|$y = x^3$|$\mathbb{R}$|$\mathbb{R}$|**奇**|$\mathbb{R}$ 上**增**|
|$y = \sqrt{x}$|$[0, +\infty)$|$[0, +\infty)$|非奇非偶|$[0,+\infty)$**增**|
|$y = \frac{1}{x}$|$(-\infty,0)\cup(0,+\infty)$|$(-\infty,0)\cup(0,+\infty)$|**奇**|$(-\infty,0)$**减**，$(0,+\infty)$**减**|

### 一般幂函数性质
1. **$\alpha > 0$ 时**：
    - 图象过 $(0,0)$ 和 $(1,1)$
    - 在 $(0,+\infty)$ 上单调递增
    - 在第一象限内，$\alpha$ 越大，图象越陡峭
2. **$\alpha < 0$ 时**：
    - 图象过 $(1,1)$
    - 在 $(0,+\infty)$ 上单调递减
    - 在第一象限内，$\alpha$ 越小（负得越多），图象越陡峭

# 二次函数
### 图象与性质
- 标准形式：$y = ax^2 + bx + c \ (a \neq 0)$
- 图象：**抛物线**
    - $a>0$：开口向上
    - $a<0$：开口向下
- 对称轴：$x = -\frac{b}{2a}$
- 顶点坐标：$\left( -\frac{b}{2a}, \frac{4ac-b^2}{4a} \right)$
- 零点：$x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$（由判别式 $\Delta = b^2-4ac$ 决定）
### 最值问题解题思路
1. 确定开口方向（$a$ 的正负）
2. 求对称轴 $x_0 = -\frac{b}{2a}$
3. 判断对称轴与区间 $[m,n]$ 的位置关系：
    - **对称轴在区间内**：顶点处取一个最值，端点取另一个最值
    - **对称轴在区间左侧**：最值在右端点 $x=n$ 处
    - **对称轴在区间右侧**：最值在左端点 $x=m$ 处
4. 计算并比较函数值
#### 最值情况分析表

|开口方向|对称轴位置|最小值位置|最大值位置|
|---|---|---|---|
|$a>0$|$x_0 \in [m,n]$|顶点 $x_0$|离 $x_0$ 较远的端点|
|$a>0$|$x_0 < m$|$x=m$|$x=n$|
|$a>0$|$x_0 > n$|$x=n$|$x=m$|
|$a<0$|$x_0 \in [m,n]$|离 $x_0$ 较远的端点|顶点 $x_0$|
|$a<0$|$x_0 < m$|$x=n$|$x=m$|
|$a<0$|$x_0 > n$|$x=m$|$x=n$|

# 指数与指数函数
### 根式与指数幂运算
1. **根式与分数指数幂转换**：
    - $a^{\frac{m}{n}} = \sqrt[n]{a^m}$（$a>0, m,n\in\mathbb{N}^*, n>1$）
    - $a^{-\frac{m}{n}} = \frac{1}{a^{\frac{m}{n}}}$
2. **指数运算性质**：
    - $a^m \cdot a^n = a^{m+n}$
    - $(a^m)^n = a^{m \cdot n}$
    - $(ab)^n = a^n b^n$
    - $a^0 = 1 \ (a \neq 0)$
    - $a^{-n} = \frac{1}{a^n}$
### 指数函数的图象与性质
- 一般形式：$y = a^x \ (a>0, a \neq 1)$
- 定义域：$\mathbb{R}$
- 值域：$(0, +\infty)$
- 图象必过点 $(0,1)$

# 对数与对数函数
### 对数运算
1. **定义**：$a^x = b \Leftrightarrow x = \log_a b \ (a>0, a\neq1, b>0)$
2. **运算性质**：
    - $\log_a 1 = 0$    
    - $\log_a a = 1$
    - $\log_a (MN) = \log_a M + \log_a N$
    - $\log_a \left(\frac{M}{N}\right) = \log_a M - \log_a N$
    - $\log_a M^n = n \log_a M$
    - 换底公式：$\log_a b = \frac{\log_c b}{\log_c a}$
    - $\log_{a^m} b^n = \frac{n}{m} \log_a b$
### 对数函数的图象与性质
- 一般形式：$y = \log_a x \ (a>0, a\neq1)$
- 定义域：$(0,+\infty)$
- 值域：$\mathbb{R}$
- 图象必过点 $(1,0)$
