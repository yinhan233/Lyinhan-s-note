# 双曲线与抛物线定理总结

## 一、双曲线

### 1. 定义
平面内到两个定点 $F_1$, $F_2$ 的距离之差的绝对值等于常数 $2a$ ($0 < 2a < |F_1F_2|$) 的点的轨迹。

### 2. 标准方程
$\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$ $(a,b > 0)$  
其中 $c = \sqrt{a^2 + b^2}$，焦点 $(\pm c, 0)$

### 3. 基本性质
- **渐近线**：$y = \pm \dfrac{b}{a}x$
- **离心率**：$e = \dfrac{c}{a} > 1$
- **焦半径公式**（右支上点 $P(x_0,y_0)$）：  
  $|PF_1| = ex_0 + a$，$|PF_2| = ex_0 - a$  
  （左支取负号）

### 4. 拓展定理及推导
#### (1) 焦点三角形面积公式
$\triangle PF_1F_2$ 面积 $S = \dfrac{b^2}{\tan \dfrac{\theta}{2}}$（$\theta = \angle F_1PF_2$）

**推导**：  
设 $|PF_1|=r_1, |PF_2|=r_2$，则 $|r_1-r_2|=2a$  
由余弦定理：  
$(2c)^2 = r_1^2 + r_2^2 - 2r_1r_2\cos\theta$  
$4c^2 = (r_1-r_2)^2 + 2r_1r_2(1 - \cos\theta)$  
$4c^2 = 4a^2 + 2r_1r_2(1 - \cos\theta)$  
$\Rightarrow r_1r_2 = \dfrac{2b^2}{1 - \cos\theta}$  
$S = \dfrac{1}{2}r_1r_2\sin\theta = \dfrac{b^2 \sin\theta}{1 - \cos\theta} = \dfrac{b^2}{\tan\frac{\theta}{2}}$

#### (2) 共轭双曲线
双曲线 $C_1: \dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$ 的共轭双曲线为：  
$C_2: \dfrac{y^2}{b^2} - \dfrac{x^2}{a^2} = 1$  
**性质**：  
- 相同渐近线 $y = \pm \dfrac{b}{a}x$  
- 四个焦点 $(\pm c,0)$, $(0,\pm c)$ 共圆：$x^2 + y^2 = a^2 + b^2$

---

## 二、抛物线

### 1. 定义
平面内到定点 $F$（焦点）与定直线 $l$（准线）距离相等的点的轨迹。

### 2. 标准方程
$y^2 = 2px$ ($p>0$)，焦点 $F(\dfrac{p}{2},0)$，准线 $x = -\dfrac{p}{2}$

### 3. 基本性质
- **离心率**：$e=1$
- **焦半径公式**：$|PF| = x_0 + \dfrac{p}{2}$（$P(x_0,y_0)$ 在抛物线上）
- **通径长**：过焦点的最短弦，长度 $= 2p$

### 4. 拓展定理及推导
#### (1) 焦点弦性质
过焦点 $F$ 的弦 $AB$（倾斜角 $\alpha$）：  
1. 弦长 $|AB| = \dfrac{2p}{\sin^2\alpha}$  
2. $\triangle AOB$ 面积 $S = \dfrac{p^2}{2\sin\alpha}$（$O$ 为顶点）

**推导**：  
设 $AB: y = k(x - \dfrac{p}{2})$，$k = \tan\alpha$  
联立 $y^2 = 2px$：  
$k^2(x - \dfrac{p}{2})^2 = 2px$  
整理得：$k^2x^2 - (k^2p + 2p)x + \dfrac{k^2p^2}{4} = 0$  
弦长 $|AB| = \sqrt{1+k^2} \cdot \dfrac{\sqrt{\Delta}}{|k^2|} = \dfrac{2p(1+k^2)}{k^2} = \dfrac{2p}{\sin^2\alpha}$  
面积 $S_{\triangle AOB} = \dfrac{1}{2} \cdot \dfrac{p}{2} \cdot |y_A - y_B| = \dfrac{p^2}{2\sin\alpha}$

#### (2) 切线方程
抛物线 $y^2 = 2px$ 上点 $P(x_0,y_0)$ 的切线：  
$y_0y = p(x + x_0)$

**推导**（导数法）：  
由 $y^2 = 2px$ 得 $2yy' = 2p \Rightarrow y' = \dfrac{p}{y}$  
在 $P(x_0,y_0)$ 处斜率 $k = \dfrac{p}{y_0}$  
切线方程：$y - y_0 = \dfrac{p}{y_0}(x - x_0)$  
化简得：$y_0y = px + y_0^2 - px_0$  
代入 $y_0^2 = 2px_0$ 即证。

## 三、统一性质对比
| 性质         | 双曲线                        | 抛物线                      |
|--------------|-------------------------------|-----------------------------|
| **定义**     | $\| \|PF_1\| - \|PF_2\| \| = 2a$ | $|PF| =$ 点P到准线距离 |
| **离心率**   | $e > 1$                       | $e = 1$                    |
| **焦点数**   | 两个                          | 一个                       |
| **光学性质** | 反射光线反向延长线过另一焦点  | 反射光线平行于对称轴       |
| **渐近线**   | 存在                          | 不存在                     |
# 双曲线与三角函数的结合

## 一、双曲线的三角函数参数方程
双曲线 $\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$ 可用三角函数参数化为：

$$
\boxed{
\begin{cases}
x = a\sec\theta \\
y = b\tan\theta
\end{cases}
}
\quad (\theta \in [0,2\pi),\ \theta \neq \frac{\pi}{2},\frac{3\pi}{2})
$$

**推导**：  
由双曲线定义 $\left(\dfrac{x}{a}\right)^2 - \left(\dfrac{y}{b}\right)^2 = 1$  
令 $\dfrac{x}{a} = \sec\theta$，则 $\left(\dfrac{y}{b}\right)^2 = \sec^2\theta - 1 = \tan^2\theta$  
取 $\dfrac{y}{b} = \tan\theta$ 即得参数方程

> **几何意义**：  
> $\theta$ 为渐近线的旋转角（如图示），当 $\theta \to \dfrac{\pi}{2}^-$ 时，$y \to +\infty$；当 $\theta \to \dfrac{\pi}{2}^+$ 时，$y \to -\infty$

---

## 二、重要性质推导

### 1. 焦半径公式（三角函数形式）
设 $P(a\sec\theta, b\tan\theta)$，离心率 $e = \dfrac{c}{a}$, $c = \sqrt{a^2 + b^2}$

右焦点 $F_2(c,0)$ 的焦半径：
$$
|PF_2| = a(e\sec\theta - 1)
$$

**推导**：  
$|PF_2| = \sqrt{(a\sec\theta - c)^2 + (b\tan\theta - 0)^2}$  
$= \sqrt{a^2\sec^2\theta - 2ac\sec\theta + c^2 + b^2\tan^2\theta}$  
由 $c^2 = a^2 + b^2$，代入得：  
$= \sqrt{a^2\sec^2\theta - 2ac\sec\theta + a^2 + b^2(\sec^2\theta - 1)}$  
$= \sqrt{(a^2 + b^2)\sec^2\theta - 2ac\sec\theta + a^2 - b^2}$  
$= \sqrt{c^2\sec^2\theta - 2ac\sec\theta + a^2}$  
$= \sqrt{(c\sec\theta - a)^2} = |c\sec\theta - a|$  
$\because \sec\theta \geq 1,\ c > a$，故 $|PF_2| = c\sec\theta - a = a(e\sec\theta - 1)$

### 2. 焦点三角形面积
$\triangle PF_1F_2$ 面积：
$$
S = b^2 \cot\frac{\theta}{2}
$$
其中 $\theta = \angle F_1PF_2$

**推导**：  
由余弦定理：  
$|F_1F_2|^2 = |PF_1|^2 + |PF_2|^2 - 2|PF_1||PF_2|\cos\theta$  
$4c^2 = (a|e\sec\theta + 1|)^2 + (a|e\sec\theta - 1|)^2 - 2|PF_1||PF_2|\cos\theta$  
（以右支点为例）  
$4c^2 = a^2(e\sec\theta + 1)^2 + a^2(e\sec\theta - 1)^2 - 2|PF_1||PF_2|\cos\theta$  
$= 2a^2(e^2\sec^2\theta + 1) - 2|PF_1||PF_2|\cos\theta$  
$\Rightarrow |PF_1||PF_2| = \dfrac{2a^2(e^2\sec^2\theta + 1) - 4c^2}{2\cos\theta}$  
代入 $c^2 = a^2e^2$ 化简得：  
$|PF_1||PF_2| = \dfrac{2a^2b^2\sec^2\theta}{c^2\cos\theta}$  
$S = \dfrac{1}{2}|PF_1||PF_2|\sin\theta = b^2 \cdot \dfrac{\sec^2\theta \sin\theta}{2(1 - \cos^2\theta)} = b^2 \cot\frac{\theta}{2}$

## 三、与椭圆的对比
| 性质         | 椭圆参数方程                  | 双曲线参数方程                |
|--------------|-------------------------------|-------------------------------|
| **参数形式** | $\begin{cases} x=a\cos t \\ y=b\sin t \end{cases}$ | $\begin{cases} x=a\sec\theta \\ y=b\tan\theta \end{cases}$ |
| **参数范围** | $t \in [0,2\pi)$              | $\theta \in [0,2\pi),\ \theta \neq \frac{\pi}{2},\frac{3\pi}{2}$ |
| **几何意义** | 离心角                        | 渐近线旋转角                  |
| **焦半径**   | $|PF_1| = a(1 - e\cos t)$   | $|PF_2| = a(e\sec\theta - 1)$ |
| **面积公式** | $S = b^2 \tan\frac{\theta}{2}$ | $S = b^2 \cot\frac{\theta}{2}$ |

> **注**：双曲线的三角函数表示在解决特定问题时极为高效，尤其涉及渐近线夹角、离心率范围等问题时优势明显。
