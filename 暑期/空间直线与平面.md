# 平面及其基本性质

## 空间的点与直线、点与平面的位置关系

| 对象关系    | 位置关系描述                     | 符号表示 |
|-------------|----------------------------------|----------|
| **点与直线** | 点 A 在直线 l 上（直线 l 经过点 A） | A ∈ l    |
|             | 点 A 不在直线 l 上（直线 l 不经过点 A） | A ∉ l    |
| **点与平面** | 点 A 在平面 α 上（平面 α 经过点 A） | A ∈ α    |
|             | 点 A 不在平面 α 上（平面 α 不经过点 A） | A ∉ α    |

## 平面的基本性质

| 名称    | 内容                                                                 | 作用             |
|---------|----------------------------------------------------------------------|------------------|
| 公理 1  | 如果一条直线有两个点在一个平面上，那么这条直线上的所有点都在这个平面上 | 判定直线在平面上 |
| 公理 2  | 不在同一直线上的三点确定一个平面                                       | 确定一个平面     |
| 推论 1  | 一条直线和该直线外的一点确定一个平面                                   | 确定一个平面     |
| 推论 2  | 两条相交直线确定一个平面                                               | 确定一个平面     |
| 推论 3  | 两条平行直线确定一个平面                                               | 确定一个平面     |
| 公理 3  | 如果两个不同的平面有一个公共点，那么它们有且只有一条过该点的公共直线     | 判定两平面相交   |

# 直线与直线的位置关系

## 平行公理
平行于同一条直线的两条直线互相平行

## 等角定理
如果两条相交直线与另外两条直线分别平行，那么这两组相交直线所成的锐角（或直角）相等

## 空间两条直线的位置关系

| 位置关系       | 细分   | 表示方法      | 公共点个数 |
|----------------|--------|---------------|------------|
| **共面直线**   | 相交   | a ∩ b = A     | 1          |
|                | 平行   | a ∥ b         | 0          |
| **异面直线**   | ——     | a, b 是异面直线 | 0          |

## 异面直线
1. **概念**：不同在任何一个平面内的两条直线
2. **判定方法**：
   - 定义法（常用反证法）
   - 异面直线判定定理：过平面外一点与平面内一点的直线，和此平面内不经过该点的直线是异面直线
3. **异面直线所成角**：通过平移转化为相交直线所成的锐角（或直角），范围 (0, π/2]
4. **公垂线与公垂线段**：
   - 公垂线：与两条异面直线都垂直且相交的直线
   - 公垂线段：公垂线在两异面直线间的线段（连接两异面直线上点的最短线段）
5. **异面直线距离**：公垂线段的长度

# 直线与平面的位置关系

| 位置关系       | 细分    | 二次细分 | 表示方法      | 公共点数量   |
| ---------- | ----- | ---- | --------- | ------- |
| **直线在平面内** | ——    |      | a ⊂ α     | 无穷多<br> |
| **直线在平面外** | 平行    |      | a ∥ α     | 0       |
|            | 与平面相交 | 斜交   | a ∩ α = A | 1       |
|            |       | 垂直   | a ⊥ α     | 1       |

## 线面平行
- **判定定理**：若平面外一条直线与此平面内的一条直线平行，则该直线与此平面平行（线线平行 ⇒ 线面平行）
- **性质定理**：若一条直线与一个平面平行，过该直线的平面与此平面相交，则交线与该直线平行（线面平行 ⇒ 线线平行）

## 线面垂直
- **判定定理**：若一条直线与一个平面内的两条相交直线都垂直，则该直线与此平面垂直
- **性质定理**：垂直于同一平面的两条直线平行
  - 推论 1：过一点有且只有一个平面与已知直线垂直
  - 推论 2：过一点有且只有一条直线与已知平面垂直

## 点到平面的距离
从平面外一点向平面作垂线，该点到垂足的距离称为**点到平面的距离**。  
**性质**：是空间点到该平面上任意一点距离的最小值。

## 三垂线定理
平面内的一条直线，如果和这个平面的一条斜线的射影垂直，那么它也和这条斜线垂直（反之亦成立）。  
**符号表述**：  
设 l 是平面 α 的斜线，l' 是 l 在 α 上的射影，若 a ⊂ α 且 a ⊥ l'，则 a ⊥ l。
# 平面与平面的位置关系
## 平面与平面的位置关系

| 位置关系       | 符号表示       | 公共点       | 图示特征               |
|----------------|----------------|--------------|------------------------|
| **两平面平行** | α ∥ β         | 无公共点      | 永不相交               |
| **两平面相交** | α ∩ β = l     | 一条公共直线  | 交线为直线             |

## 平面平行的判定与性质
### 判定定理
如果一个平面内的**两条相交直线**分别平行于另一个平面，则这两个平面平行。  
（线面平行 ⇒ 面面平行）
### 性质定理
若两个平行平面同时与第三个平面相交，则它们的**交线相互平行**。  
（面面平行 ⇒ 线线平行）
## 二面角
### 定义
从一条直线出发的两个**半平面**所组成的图形
### 二面角的平面角
以二面角的棱上任意一点为端点，在两个半平面内分别作**垂直于棱**的两条射线所成的角。  
**性质**：
- 平面角大小与棱上点的位置无关
- 范围：[0, π]
## 平面与平面垂直
### 判定定理
如果一个平面经过另一个平面的**一条垂线**，则这两个平面互相垂直。  
（线面垂直 ⇒ 面面垂直）  
符号表示：若 l ⊥ α 且 l ⊂ β，则 α ⊥ β
### 性质定理
若两个平面互相垂直，则在一个平面内**垂直于交线**的直线，必垂直于另一个平面。  
（面面垂直 ⇒ 线面垂直）  
符号表示：若 α ⊥ β，α ∩ β = l，且 a ⊂ α，a ⊥ l，则 a ⊥ β
### 性质推论
1. 若两平面垂直，则过第一个平面内一点且垂直于第二个平面的直线，必在第一个平面内
2. 两相交平面同时垂直于第三个平面，则它们的交线也垂直于第三个平面
